  • Captain with Jump Pack — [declare] Garder 1CP pour relance ; mesurer contre-charge ennemie.

  • Terminator Squad — [declare] Menacer via rerolls alliés ; multi-charge si possible.
Phase de Combat
  • Rappels généraux :
  • Activer dans le bon ordre ; Pile-in/Consolidate pour voler OC.
  • Captain with Jump Pack — [consolidate] Consolider pour voler OC.
  • Chaplain — [make_attacks] Activer au bon timing pour maximiser +1 touche.

  • Bladeguard Veteran Squad — [make_attacks] Choisir ordre d’activation pour sécuriser.
  • Terminator Squad — [make_attacks] Frapper priorités ; consolider pour voler OC.
Fin de tour
  • Rappels généraux :
  • Compter OC ; scorer primaires/secondaires ; valider actions.
